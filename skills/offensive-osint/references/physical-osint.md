# Physical OSINT — Advisory Only

This file exists to **remind operators that physical OSINT is
out-of-scope for typical external bug-bounty / pentest engagements**.

## What CBH will NOT help with

- Tailgating / badge cloning / lock picking — not a software issue
- Identifying employees from photos for social engineering — high
  risk of harassment claims
- Physical surveillance / "case the building"
- Dumpster diving

## Why this matters in a knowledge artifact

Operators sometimes try to use CBH for adjacent surface work. The
honest answer is: a Claude Code skill bundle is the wrong tool. If
your engagement scope includes physical, hire a physical pentest
firm and operate under their methodology.
